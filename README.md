# cn-poker-archive

本仓库用于归档和发布多个独立 HTML 页面，主要面向中文扑克相关内容的整理与存储。仓库内容不针对任何单个域名或具体网站，仅为页面文件的集中存放与版本管理提供便利。

## 项目简介

cn-poker-archive 是一个以归档为目的的静态页面仓库，收录并维护多个与扑克主题相关的 HTML 文件。这些文件可独立运行，适合用于资料查阅、历史记录保存或离线浏览。

## 目录说明

```
cn-poker-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的静态资源（样式、脚本、图片等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立的归档页面，命名清晰，便于识别。
- `assets/`：存放页面共用的或独立的资源文件，保持归档文件的自包含性。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，可直接在浏览器中打开查看。
- 页面内容以中文为主，涵盖扑克规则、历史记录、工具页面等。
- 归档文件不依赖外部在线服务，旨在提供稳定、可复现的浏览体验。
- 页面更新会通过 Git 提交记录进行追踪，便于回溯历史版本。

## 维护说明

- 本仓库由维护者不定期更新，欢迎提交 Issue 或 Pull Request 提供建议或补充内容。
- 提交新页面时，请确保文件结构清晰，资源文件放置于 `assets/` 对应子目录。
- 请勿在页面中包含恶意代码、广告或违反平台政策的内容。
- 仓库内容仅供存档与学习交流，使用者应自行遵守相关法律法规。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/cn-poker-archive.git
   ```
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件，或通过 `index.html` 浏览索引。
3. 也可通过 GitHub Pages 等服务在线访问（需自行配置）。
